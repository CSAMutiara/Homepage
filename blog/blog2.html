more info about context



Contextual Colors:


"Primary" text is blue

"Successful" text is green

"Informational" text is turquoise

"Warning" text is yellow

"Dangerous" text is red

"Muted" text is grey


Contextual Background Colors:

"Primary" text gets a background that is blue

"Successful" text gets a background that is green

"Informational" text gets a background that is turquoise

"Warning" text gets a background that is yellow

"Dangerous" text gets a background that is red

"Muted" text doesn't get any special background ):
